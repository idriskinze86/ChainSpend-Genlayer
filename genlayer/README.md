# ChainSpend × Expense Court

Expense Court is a proposed GenLayer Intelligent Contract that helps resolve expense disputes using AI reasoning.

## Problem

Expense reimbursement often leads to disagreements:

- "I paid for dinner."
- "No, we split the bill."
- "This receipt is fake."

Traditional smart contracts cannot evaluate evidence.

GenLayer Intelligent Contracts can.

## Solution

Expense Court allows users to submit evidence.

The Intelligent Contract reasons over:

- receipts
- timestamps
- descriptions
- participant statements

before returning a recommendation.

This demonstrates how ChainSpend can evolve into an intelligent financial application.
